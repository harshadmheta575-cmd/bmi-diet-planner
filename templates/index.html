About This Project

This BMI Based Diet Planner is a health-focused web application that calculates Body Mass Index and generates personalized weekly diet plans based on BMI category and food preference (Veg / Non-Veg).

It helps users understand daily nutrition requirements, protein intake, and healthy lifestyle recommendations in a simple and interactive way.
